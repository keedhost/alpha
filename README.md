# Alpha

Status-bar додаток для macOS, який автоматично виправляє розкладку введення між українською та англійською мовами.

## Функціонал

- Іконка в системному треї у вигляді грецької літери "Αльфа".
- Перемикач автокорекції: "Увімкнути автокорекцію" / "Вимкнути автокорекцію".
- Автоматичне перетворення введеного слова між розкладками, якщо воно набране не в тій мові.
- Звук клацання при виконанні автокорекції.
- Вікно "Налаштування" з опцією запуску під час входу в систему.
- Вікно "Про програму" з версією, автором, ліцензією GNU GPL, посиланням на проєкт і заявою про приватність.

## Збірка з вихідного коду

Вимоги:
- macOS 13 або новіша
- Xcode 15 або новіша

Збірка командою:

```bash
./build.sh
```

Проєкт можна також відкрити у Xcode:

```bash
open Alpha/Alpha.xcodeproj
```

Після першого запуску додаток попросить доступ до "Доступність" (Accessibility) для корекції вводу.

## Локалізація

Проєкт використовує стандартні `.strings` файли для локалізації:
- `Alpha/Alpha/en.lproj/Localizable.strings`
- `Alpha/Alpha/uk.lproj/Localizable.strings`

Мову інтерфейсу можна змінити у вікні **Налаштування**. За замовчуванням вона встановлюється за системною мовою (українська для `uk`, інакше англійська).
